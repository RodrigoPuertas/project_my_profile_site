# Design System: Kinetic Architect (Cyber-Editorial)

This design system is anchored in the Stitch project ID `9270569860920448406`. All future generations in this project will automatically inherit these styles.

## Visual North Star
**"The Kinetic Architect"**: A high-end, cyber-editorial aesthetic for technical portfolios. Deep dark modes, intentional asymmetry, and luminous glass layers.

## Technical Tokens

### Colors (Midnight Void)
- **Background**: `#0b0e14`
- **Primary (Electric Indigo)**: `#a7a5ff`
- **Secondary (Cyber Violet)**: `#af88ff`
- **Tertiary (Quantum Blue)**: `#8ce7ff`
- **Surface**: `#0b0e14` (Tier 0)
- **Surface Container Low**: `#10131a` (Tier 1)
- **Surface Container Highest**: `#22262f` (Tier 2 - Cards)

### Typography
- **Headlines**: Space Grotesk (All-caps, high tracking for titles)
- **Body/Labels**: Inter (Clean, high-precision technical text)

### Principles
1. **No-Line Rule**: No solid 1px borders for layout. Use background color shifts to define sections.
2. **Glassmorphism**: Use `backdrop-blur(20px)` on floating elements with low-opacity surfaces.
3. **Ghost Borders**: Use `outline-variant` at 15% opacity for subtle edge definition.
4. **Asymmetry**: Break the centered grid with large display type and offset cards.
